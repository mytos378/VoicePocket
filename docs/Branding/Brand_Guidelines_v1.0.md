# VoicePocket Brand Guidelines v1.0

Versión: 1.0

Estado: Oficial

---

# Propósito

Este documento define la identidad visual oficial de VoicePocket.

Toda decisión relacionada con el diseño deberá respetar estas directrices para mantener una experiencia consistente y reconocible.

---

# Esencia de la Marca

VoicePocket representa el instante en que una idea encuentra un lugar seguro.

No es una aplicación para escribir notas.

Es un lugar donde nacen las ideas.

---

# Concepto Oficial

Pocket Spark

El logotipo representa una chispa descendiendo hacia un bolsillo.

La chispa simboliza:

• inspiración

• creatividad

• conocimiento

• ideas

El bolsillo simboliza:

• seguridad

• cercanía

• disponibilidad

• memoria

---

# Misión

Capturar una idea antes de que desaparezca.

---

# Eslogan

Principal

Nunca pierdas una idea.

Secundario

Las mejores ideas duran segundos.
Guárdalas para siempre.

---

# Personalidad

VoicePocket debe sentirse:

✔ rápido

✔ simple

✔ confiable

✔ profesional

✔ silencioso

✔ organizado

Nunca deberá sentirse:

✘ infantil

✘ complejo

✘ agresivo

✘ tecnológico en exceso

✘ futurista

---

# Valores

Rapidez.

Simplicidad.

Privacidad.

Confiabilidad.

Minimalismo.

---

# Identidad Visual

El protagonista visual siempre será el concepto Pocket Spark.

El logotipo nunca deberá modificarse conceptualmente.

Únicamente podrán ajustarse detalles de ejecución:

• grosor

• proporciones

• color

• adaptación de tamaño

Nunca deberá sustituirse la chispa por un micrófono, una bombilla o un cerebro.

---

# Filosofía de Diseño

Menos elementos.

Más claridad.

Más espacio.

Menos distracciones.

Más ideas.

---

# Principio Fundamental

La interfaz nunca deberá competir con las ideas del usuario.

El contenido siempre será el protagonista.
